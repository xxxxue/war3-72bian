# war3-72bian
War3-官方对战平台RPG地图-七十二变之斩妖除魔 修改器

## 使用方法

选择角色后, 点击软件中相应的功能即可, 

`注意: 先点击 等级5和25 解锁技能, 技能解锁后 , 再点击等级255 `

## 游戏版本:

War3版本:  `1.27.0.52240`

地图版本: `2.0.0`
